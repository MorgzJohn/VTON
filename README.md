# VTON — Virtual Try-On System Research

## Overview
Research implementation of IDM-VTON for MCA final year project.
Evaluating AI-based virtual garment try-on using diffusion models.

## Model
- IDM-VTON (yisol/IDM-VTON)
- Diffusion-based virtual try-on
- GPU: Kaggle T4 (15GB)
